# Git-AI Commit Automation

Alat bantu CLI kustom untuk mengotomatisasi pembuatan pesan *commit* Git menggunakan Google Gemini API. Alat ini dirancang untuk bekerja langsung di PowerShell untuk efisiensi *workflow*.

## 🚀 How to Setup

Ikuti langkah-langkah berikut untuk mengintegrasikan `git-ai` ke dalam terminal PowerShell Anda:

1. buka setting pada CLI vs code anda `notepad $PROFILE`
2. lalu copy dan pastekan teks pada code.md
3. simpan file `ctrl + s`
4. lalu tutup dengan `. $PROFILE`

Setup selesai, jalankan `git-ai` lalu tekan y/n untuk hasil nya lalu enter.
